### What This Program Does

---
***This program takes a simple arithmetic expression in the form: x y z *** 
- Example: 1 + 2, where x = 1, y = + and z = 2

It calculates the result and outputs it as a float with one decimal place.

### How to Test the Program

---
***Run the program with python interpreter.py***
- Expected Outputs -->
                            1 + 1     → 2.0
                            10 - 3    → 7.0
                            2 * 4     → 8.0
                            10 / 4    → 2.5

 ---
 > The program splits the input into parts, converts numbers, performs the operation, and formats the result.

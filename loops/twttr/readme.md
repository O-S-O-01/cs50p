### What This Program Does

---

***Removes all vowels (A, E, I, O, U) from a string, regardless of case.***

---
### How to Test the Program

Run the program with python3 twttr.py

```
Examples:

Input: Twitter
Output: Twttr

Input: HELLO
Output: HLL

Input: cs50
Output: cs50
```
---

>Loop through characters Use .lower() to handle uppercase Keep only non-vowel characters

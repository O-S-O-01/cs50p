### What This Program Does

---

***Converts a variable name from camelCase to snake_case by inserting underscores before uppercase letters and converting them to lowercase.***

---
### How to Test the Program
Run the program with python3 camel.py

    Examples:

    Input: firstName

    Output: first_name

    Input: preferredFirstName

    Output: 

    preferred_first_name

---
>Loop through each character Detect uppercase letters using .isupper() Build a new string using +=

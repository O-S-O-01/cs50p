### What This Program Does

---
***Takes a fruit name and prints its calories using a dictionary lookup.***

---
### How to Test the Program

Run the program with python3  nutrition.py

    Examples:

    Fruit: Apple
   
    130

    Fruit: banana
    
    110

    Invalid input:

    Fruit: mango

    (no output)

>Use a dictionary for fast lookup Convert input to lowercase Check existence using if fruit in fruits

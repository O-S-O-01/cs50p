### What This Program Does

---

***Simulates a vending machine that accepts coins (25, 10, 5) until at least 50 cents is inserted, then returns change.***

---
### How to Test the Program

Run the program with python3 coke.py
```
    Example:

    Insert coin: 25
    Amount due: 25
    Insert coin: 25
    Change owed: 0
                            ```

Invalid coin:

    Insert coin: 3

    Invalid coin, try again.
